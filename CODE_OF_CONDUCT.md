# Code of Conduct

## Our standard

Stealth Trails Bank is a professional collaboration space.

We want this repository to be a place where people can work clearly, respectfully, and productively. Disagreement is normal. Disrespect is not.

## Expected behavior

Everyone contributing to this repo is expected to:

- communicate respectfully
- assume good intent before jumping to conclusions
- give feedback on the work, not the person
- be open to review and correction
- keep discussions focused, useful, and professional
- handle disagreements calmly
- respect boundaries around time, ownership, and decision-making

## Unacceptable behavior

The following are not acceptable in this repository or related collaboration spaces:

- harassment or intimidation
- insulting, mocking, or demeaning language
- personal attacks
- discrimination or exclusionary behavior
- repeated bad-faith argument
- intentionally disruptive behavior
- sharing private or sensitive information without permission
- pressuring others to ignore security, quality, or review standards

## Review culture

Code review is part of the product process here.

That means:

- contributors should expect honest technical feedback
- reviewers should give direct but respectful guidance
- both sides should focus on correctness, clarity, and product safety

## Maintainer responsibility

Maintainers are responsible for:

- keeping collaboration professional
- setting direction when decisions are needed
- enforcing repo standards fairly
- stepping in when behavior harms the team or the work

## Reporting concerns

If someone is making collaboration unsafe or unworkable, report it privately to the maintainers or repository owners.

Please include:

- what happened
- where it happened
- who was involved
- any useful context

Reports will be handled as carefully and discreetly as possible.

## Consequences

Unacceptable behavior may result in actions including:

- a warning
- removal from a discussion
- restriction from contributing
- broader repository access removal

The response will depend on severity and pattern of behavior.

## Final note

We want a repo culture that is calm, direct, and serious about the product.

That means being respectful is not separate from doing good work here. It is part of doing good work.
